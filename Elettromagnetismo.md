# Forza magnetica
$$\begin{flalign}\vec{F_{m}}=\frac{\mu_{0}}{4\pi}\cdot \frac{q_{1}\vec{v_{1}}\times(q_{2}\vec{v_{2}}\times \hat{u_{r}})}{r^{2}} &&\end{flalign}$$

# Campo magnetico
$$\begin{flalign}\vec{B}=\frac{\mu_{0}}{4\pi}\cdot \frac{q_{2}\vec{v_{2}}\times \hat{u_{r}}}{r^{2}} &&\end{flalign}$$

# Campo magnetico di un filo infinito
$$\begin{flalign}B=\frac{\mu_{0}I}{2\pi r} &&\end{flalign}$$

# Solenoide
$$\begin{flalign}B=I \frac{n}{l}\mu_{0}=IN\mu_{0} &&\end{flalign}$$

# Forza di Lorentz
$\vec{F}=q\cdot(\vec{E}+\vec{v}\times \vec{B})$
### Su un filo
$F=I\cdot L\cdot B\cdot \sin(\theta)$

# Equazioni di Maxwell
### Gauss
$$\begin{flalign}\oint_{S}\vec{E}\cdot d\vec{s}=\frac{Q}{\varepsilon_{0}} &&\end{flalign}$$

### Faraday
$$\begin{flalign}\oint_{l} \vec{E}\cdot d\vec{l}=-\frac{d}{dt}\int_{S}\vec{B}\cdot d\vec{s} &&\end{flalign}$$

### Ampere
$$\begin{flalign}\oint_{l}\vec{B}\cdot d\vec{l}=\mu_{0}I+\mu_{0}\varepsilon_{0} \frac{d}{dt}\int_{S}\vec{E}\cdot d\vec{s} &&\end{flalign}$$

### Gauss
$$\begin{flalign}\oint_{S}\vec{B}\cdot d\vec{s}=0 &&\end{flalign}$$
